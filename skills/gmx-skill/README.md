# gmx-skill

Perpetual futures trading, GM token liquidity provision, and market data on GMX V2 (Arbitrum and Avalanche).

## Install
```bash
bash install.sh
.\install.ps1
```

## Keys required
```
EVM_PRIVATE_KEY=...     # from ~/.jelly-claude/.keys
ARB_RPC_URL=...         # Arbitrum RPC (Alchemy / public)
```

## Used by agents
- `cross-exchange-arb`, `defi-yield-optimizer`
