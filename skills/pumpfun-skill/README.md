# pumpfun-skill

AI coding skill for PumpFun Protocol covering token launches, bonding curves, and PumpSwap integrations.

## Install (upstream recommended)
```bash
npx skills add https://github.com/sendaifun/skills
```

Or local:
```bash
bash install.sh
```
