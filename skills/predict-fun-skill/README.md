# predict-fun-skill v2

Teach your Jelly agent to browse, analyze, and trade on **[predict.fun](https://predict.fun)** — a live CLOB binary prediction market on BNB Chain, settled in USDT.

## What's new in v2

| Feature | v1 | v2 |
|---------|----|----|
| Auth examples | TypeScript only | TypeScript + Python |
| API coverage | Core endpoints | All 28 endpoints incl. OAuth |
| Orderbook | Basic price read | Deep analysis, complement formula, depth viz |
| Order types | LIMIT + MARKET | LIMIT + MARKET + SELL + position closing |
| OAuth connections | Not covered | Full OAuth order + position management |
| Search | Not covered | Full-text search across markets + categories |
| Tags | Not covered | Tags API |
| Timeseries | Not covered | Historical + latest value |
| Last sale | Not covered | Last trade price and size |

## What predict.fun is

predict.fun is the primary BNB Chain prediction market. Markets are binary YES/NO contracts that resolve via the UMA Optimistic Oracle. You trade by posting limit or market orders against a Central Limit Order Book (CLOB), settled in USDT.

## Prerequisites

| Requirement | How to get it |
|-------------|--------------|
| BNB Chain wallet | Auto-generated by `setup.sh` — check `~/.jelly-claude/.keys` for `EVM_PRIVATE_KEY` |
| BNB for gas | ~0.01 BNB — send from any exchange |
| USDT on BNB Chain | Buy on Binance, withdraw to BNB Chain, or swap on PancakeSwap |
| predict.fun API key | Discord → https://discord.gg/predictdotfun → `#support-ticket` → request API key |
| Node.js 18+ | For the TypeScript SDK |
| Python 3.10+ | Optional — for the Python SDK |

> **No API key needed on testnet.** Use `api-testnet.predict.fun` to practice for free.

## Add your keys

```bash
cp .keys.example ~/.jelly-claude/.keys
# Fill in:
#   EVM_PRIVATE_KEY       — your BNB Chain wallet private key
#   PREDICT_API_KEY       — from the Discord support ticket
#
# Smart Wallet (optional):
#   PRIVY_WALLET_PRIVATE_KEY  — export from predict.fun/account/settings
#   PREDICT_ACCOUNT_ADDRESS   — your deposit address
```

## Set one-time approvals

```typescript
import { Wallet } from "ethers";
import { OrderBuilder, ChainId } from "@predictdotfun/sdk";

const signer  = new Wallet(process.env.EVM_PRIVATE_KEY!);
const builder = await OrderBuilder.make(ChainId.BnbMainnet, signer);
await builder.setApprovals(); // sends 4 approval txs, needs ~0.01 BNB gas
```

Or ask the agent: **"Set up my predict.fun approvals"**
