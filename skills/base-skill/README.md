# base-skill

Interact with Base chain — swap on Aerodrome and Uniswap V3, manage LP positions, bridge from Ethereum, and read on-chain data.

## Install
```bash
bash install.sh       # Mac / Linux
.\install.ps1         # Windows PowerShell
```

## Keys required
Uses your EVM wallet auto-generated by `setup.sh`:
```
~/.jelly-claude/wallets/evm.json
```
No additional API key needed.

## Used by agents
- `base-dex-trader`
