# birdeye-skill

Token analytics for Solana and multi-chain — trending tokens, holder analysis, top traders, wallet P&L, and new listings via Birdeye API.

## Install
```bash
bash install.sh       # Mac / Linux
.\install.ps1         # Windows PowerShell
```

## Keys required
Get a free key at https://birdeye.so and add to `~/.jelly-claude/.keys`:
```
BIRDEYE_API_KEY=your_key_here
```

## Used by agents
- `birdeye-analyst`
