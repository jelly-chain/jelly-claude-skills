# uniswap-skill

Token swaps, pool data, and liquidity management on Uniswap V3/V4 across Ethereum, Base, Arbitrum, and Polygon.

## Install
```bash
bash install.sh
.\install.ps1
```

## Keys required
```
EVM_PRIVATE_KEY=...      # from ~/.jelly-claude/.keys
ETH_RPC_URL=...          # Alchemy / Infura / public RPC
```

## SDK
```bash
npm install @uniswap/v3-sdk @uniswap/sdk-core ethers
```
