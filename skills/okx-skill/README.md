# okx-skill

Spot trading, perpetuals, market data, wallet operations, and DEX aggregation via the OKX API.

## Install
```bash
bash install.sh
.\install.ps1
```

## Keys required
```
OKX_API_KEY=...
OKX_SECRET_KEY=...
OKX_PASSPHRASE=...
```
Get keys: https://www.okx.com/account/my-api

## Used by agents
- `cross-exchange-arb`, `portfolio-rebalancer`
