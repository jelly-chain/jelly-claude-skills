# solana-wallet-skill

Solana keypair generation, balance checks, devnet airdrops, and SOL transfers.

## Install
```bash
bash install.sh
```

## Required keys
None required for read-only operations. For mainnet transactions, the wallet JSON is at `~/.jelly-claude/wallets/solana.json`.

## Example prompts
- "Check my Solana wallet balance"
- "Send 0.1 SOL to [address]"
- "Generate a new Solana keypair and show me the address"
- "Airdrop 2 SOL to my devnet wallet"
