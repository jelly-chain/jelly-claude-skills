# prediction-skill

Market prediction heuristics, Jelly Score pattern, and trend confirmation logic.

## Install
```bash
bash install.sh
```

## What it does
Teaches the agent to calculate conviction scores and confirm market trends using a consistent, repeatable framework.

## Example prompts
- "Calculate the Jelly Score for this token"
- "Is SOL in a bull trend right now?"
- "What position size should I take?"
