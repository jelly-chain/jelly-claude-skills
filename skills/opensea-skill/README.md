# opensea-skill

NFT collections, listings, offers, sales history, and order execution via the OpenSea API v2.

## Install
```bash
bash install.sh
.\install.ps1
```

## Keys required
```
OPENSEA_API_KEY=...    # docs.opensea.io/reference/api-keys
EVM_PRIVATE_KEY=...    # for buying/selling NFTs on-chain
```

## Used by agents
- `nft-flipper`
