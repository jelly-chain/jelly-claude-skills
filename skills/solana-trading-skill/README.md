# solana-trading-skill

Jupiter swaps, Raydium CLMM/AMM, slippage guidelines, and pre-trade safety checks for Solana.

## Install
```bash
bash install.sh
```

## Example prompts
- "Swap 1 SOL for USDC using Jupiter"
- "Set up a DCA to buy 0.1 SOL of BONK every day for 7 days"
- "Check if this token is safe to trade"
- "Open a Raydium CLMM position on the SOL/USDC pool"
