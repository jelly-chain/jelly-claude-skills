# bnb-trading-skill

PancakeSwap v3 swaps, Venus lending, token approval patterns, and pre-trade safety checks on BNB Chain.

## Install
```bash
bash install.sh
```

## Example prompts
- "Swap 0.1 BNB for USDT on PancakeSwap"
- "Is this BSC token safe to trade? Run a honeypot check"
- "Supply 1 BNB to Venus and borrow USDT"
- "Approve PancakeSwap to spend my BUSD"
