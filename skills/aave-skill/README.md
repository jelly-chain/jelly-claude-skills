# aave-skill

Lending, borrowing, flash loans, and yield on Aave V3 across Ethereum, Base, Arbitrum, Polygon, and Avalanche.

## Install
```bash
bash install.sh
.\install.ps1
```

## Keys required
```
EVM_PRIVATE_KEY=...
ETH_RPC_URL=...
```

## SDK
```bash
npm install @aave/core-v3 @aave/contract-helpers @aave/math-utils ethers
```
