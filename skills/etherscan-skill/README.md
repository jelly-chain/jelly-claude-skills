# etherscan-skill

On-chain data for Ethereum, BNB Chain, Polygon, Base, Arbitrum, and more via Etherscan-compatible APIs.

## Install
```bash
bash install.sh
.\install.ps1
```

## Keys required
```
ETHERSCAN_API_KEY=...    # etherscan.io/apis
BSCSCAN_API_KEY=...      # bscscan.com/apis
POLYGONSCAN_API_KEY=...  # polygonscan.com/apis
```

## Used by agents
- `onchain-whale-tracker`, `token-security-auditor`, `wallet-watcher`
