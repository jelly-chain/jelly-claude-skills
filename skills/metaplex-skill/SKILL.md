# Metaplex Skill (Community Skill)

Core NFTs, Token Metadata, Bubblegum, and Candy Machine via official Metaplex SDK.

## Source
Install the official upstream version:
```bash
npx skills add https://github.com/metaplex-foundation/skill
```

View source: https://github.com/metaplex-foundation/skill

## Usage
This is a pointer skill. Install from upstream for the latest content.
The `install.sh` in this folder copies this pointer file for local reference.
