# dexscreener-skill

Discover new token pairs, monitor prices, and find trending tokens across all chains via DexScreener's public API.

## Install
```bash
bash install.sh       # Mac / Linux
.\install.ps1         # Windows PowerShell
```

## Keys required
None — DexScreener is fully public. No API key needed.

## Used by agents
- `dexscreener-scanner`

## API docs
https://docs.dexscreener.com/api/reference
