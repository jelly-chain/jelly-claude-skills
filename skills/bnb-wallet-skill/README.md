# bnb-wallet-skill

EVM wallet creation, BNB/MATIC balance checks, and BEP-20 token transfers for BNB Chain and Polygon.

## Install
```bash
bash install.sh
```

## Required
The EVM wallet is generated by `jelly-claude/setup.sh`. Private key is at `~/.jelly-claude/wallets/evm.json`.

## Example prompts
- "Check my BNB wallet balance"
- "Send 0.01 BNB to 0x..."
- "Transfer 100 USDT on Polygon to 0x..."
- "Generate a new EVM wallet and show me the address"
