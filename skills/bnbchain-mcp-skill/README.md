# bnbchain-mcp-skill

BNB Chain MCP server — blocks, contracts, tokens, NFTs, ERC-8004 agent registration, Greenfield storage.

## Install (official)
```bash
npx skills add bnb-chain/bnbchain-skills
```

Or local pointer:
```bash
bash install.sh
```

Then install the MCP server:
```bash
npx @bnb-chain/mcp@latest
```

Source: https://github.com/bnb-chain/bnbchain-skills
