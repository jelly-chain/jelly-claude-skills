# binance-skills-hub

Binance market data, spot trading, and token security audits via Binance Skills Hub.

## Install (official)
```bash
npx skills add binance/binance-skills-hub
```

Or local:
```bash
bash install.sh
```

## API key (for trading only)
Get keys at [binance.com](https://binance.com) → Profile → API Management.
Store at `~/.claude/skills/binance-skills-hub/.keys`.
