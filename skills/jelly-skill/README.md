# jelly-skill

Live DeFi data from jellychain.fun — chain TVL rankings, 24h DEX trading volume, protocol leaderboards, and trending movers. No API key required.

## Install
```bash
bash install.sh       # Mac / Linux
.\install.ps1         # Windows PowerShell
```

## Keys required
None — all jellychain.fun API endpoints are public.

## Used by agents
- `jelly-predictions-agent`

## API base
https://jellychain.fun/api
