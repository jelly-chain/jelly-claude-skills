# Testing Strategy (Solana Foundation Official Skill)

A testing pyramid for Solana programs using LiteSVM for fast unit tests, Mollusk for isolated instruction checks, and Surfpool for integration tests.

## Source
This skill is maintained by the Solana Foundation.

Install the latest version directly:
```bash
npx skills add https://github.com/solana-foundation/solana-dev-skill
```

Or view the source:
